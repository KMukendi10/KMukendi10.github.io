Case Studies

Portfolio Website

This project focused on designing and building my personal portfolio website from scratch using HTML and CSS. The goal was to create a clean and professional platform where I could showcase my skills, projects, and learning journey. I began by planning the structure of the website, including sections such as the hero, about me, projects, case studies, and contact information. Careful attention was given to spacing, typography, and layout to ensure the design remained visually consistent.

During development, I applied responsive design techniques to make sure the website worked well on different screen sizes. I also used Git to track progress and manage updates to my code. This project taught me how to organize files, structure layouts, and create reusable components. Completing this project increased my confidence in building full website layouts independently.

Tools Used: HTML, CSS, Git, GitHub, Chrome DevTools

Skills Gained: Layout Design, Responsive Design, File Organization, Version Control

Challenges & Solutions: One challenge I faced was aligning elements properly across sections. I solved this by using Flexbox and adjusting margins and spacing until the layout behaved consistently across different screen sizes.

Responsive Design

This case study focuses on building layouts that adjust automatically to different screen sizes. The objective was to ensure that web pages remained readable and visually appealing on mobile devices, tablets, and desktop computers. I used CSS media queries to change layouts based on screen width, allowing elements to stack vertically on smaller screens.

To improve layout flexibility, I used percentage widths and flexible containers. Images were resized dynamically, and navigation links were adjusted to fit smaller screens. This project helped me understand the importance of designing websites that provide a consistent user experience regardless of device size.

Tools Used: HTML, CSS, Media Queries, Flexbox, Chrome DevTools

Skills Gained: Mobile Optimization, Layout Adaptation, Media Query Implementation

Challenges & Solutions: A major challenge was keeping elements aligned when the screen size changed. I solved this by testing multiple breakpoints and adjusting layout properties such as width, padding, and alignment.

Version Control with Git

This case study demonstrates how I used Git to manage code changes and maintain version control throughout development. The main objective was to keep track of project updates and safely store versions of my work. I initialized repositories, created commits, and connected my local projects to GitHub repositories.

Using Git helped me maintain an organized development workflow. I was able to update projects regularly and keep track of previous versions when making changes. This improved my confidence in managing code professionally and understanding how developers collaborate on shared projects.

Tools Used: Git, GitHub, Command Line, VS Code

Skills Gained: Version Control, Repository Management, Commit Tracking

Challenges & Solutions: I encountered errors when pushing code to remote repositories. I resolved these issues by configuring remote URLs correctly and reviewing Git command documentation to understand the workflow better.

Troubleshooting and Debugging

This case study highlights the troubleshooting techniques I used to fix layout and styling issues during development. While working on projects, I encountered problems such as missing images, incorrect spacing, and layout misalignment. To identify the cause of these issues, I used browser developer tools to inspect elements and review applied styles.

Through repeated testing and adjustments, I improved my ability to identify errors quickly and apply effective fixes. I learned how small changes in CSS properties can significantly affect layout behavior. This process strengthened my debugging skills and helped me develop patience and attention to detail when solving technical problems.

Tools Used: HTML, CSS, Chrome DevTools, VS Code

Skills Gained: Debugging, Troubleshooting, Logical Thinking, Attention to Detail

Challenges & Solutions: One issue involved images not displaying due to incorrect file paths. I fixed this by reviewing folder structures and correcting the file references, which restored the images successfully.
